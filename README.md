# Portfolio Website Frontend

A modern and responsive personal portfolio website built using React/Next.js and TypeScript.  
This project showcases my skills, projects, certifications, and contact information in a professional way.

## Live Demo
[View Live Website](https://portfolio-website-ravisaini0s-projects.vercel.app/)

## Features

- Responsive portfolio design
- Modern UI with clean layout
- Projects showcase section
- Certificates and achievements section
- Contact form integration
- Smooth user experience
- Professional developer portfolio structure

## Tech Stack

- React / Next.js
- TypeScript
- Tailwind CSS
- HTML5
- CSS3
- JavaScript
